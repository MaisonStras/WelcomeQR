Checklist d'arrivée

Toutes les étapes pour entrer et vous installer en moins de 5 minutes.

  1. Regardez le message d'arrivée et récupérez le code (envoyé par e-mail/SMS).
  2. Saisir le code sur le boîtier : 2580 → valider → ouvrir la porte.
  3. Clé : récupérer la clé dans la boîte aimantée sous le paillasson / emplacement indiqué.
  4. Entrer : allumer la lumière, vérifier rapidement l'état général (eau chaude, propreté).
  5. Wi-Fi : connectez-vous au SSID indiqué sur cette page (section Wi-Fi).
En cas de problème d'accès : contactez le propriétaire via le bouton 'Contact propriétaire' en haut de page.
Contrôle express (30s)
  • Eau chaude OK
  • Lit & linge en ordre
  • Pas de dommages visibles

Règlement intérieur

Merci de respecter ces quelques règles pour un séjour agréable.

  • Silence après 22h — respectez le voisinage.
  • Interdit de fumer à l'intérieur.
  • Animaux acceptés uniquement sur demande.
  • Merci de trier vos déchets (bio / recyclables / ordures).
Astuce : laissez un résumé visuel (pictos) affiché dans le logement pour rappeler les règles.
Un règlement complet peut être fourni sur demande (ex. PDF ou lien Airbnb).
